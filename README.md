## Automata Rest API

Rest API for a Non-Deterministic Finite Automata based on the regular expression: a* + (b + c)*


## Endpoints

| Path                                   | Description                                                                                                                                                          |
|----------------------------------------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| /validate [GET]                    | Request the automata's word, validation and the stepwise. Get them providing *?word=* parameter.|
